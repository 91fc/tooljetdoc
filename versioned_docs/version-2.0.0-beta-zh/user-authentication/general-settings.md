## 启用注册

您可以启用/禁用 `启用注册` 。如果启用，将为用户首次 SSO 登录创建新帐户，否则仅允许现有用户通过 SSO 登录。

## 允许的域

您可以为 SSO 登录设置允许的域，可以添加多个域以逗号分隔。默认允许所有域。

## 登录网址

您可以使用登录 URL 直接登录到工作区。如果禁用多工作区，这将被隐藏。
